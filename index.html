Les modules en JS

AMD

    var $ = require('jquery');

var $ = require('jquery');

Les modules ES6


import $ from 'jquery';

const version = $.jquery;
export default version;

JSPM - Package Manager

  • Télécharger une version spécifique
  • Quelque soit son système de module utilisé
  • Depuis Github, npm ou le registry jspm

System JS - Module Loader

  • Charge les modules de façon asynchrone
  • Quelque soit le système de module utilisé
  • Système de plugins
  • Base sur polyfill d'ES6

Utiliser JSPM

Installer JSPM


						npm install jspm --save-dev

Initialiser le projet


jspm init

Installer des dépendances

L'installation des dépendances peut être faite via:

  • registry
  • github
  • npm

jspm install angular
jspm install github:samouss/angular-alert-banner
jspm install npm:moment
- JSPM registry: map keywork sur un provider spécifique (angular -> github:angular/bower-angular) - Cela permet aussi de récupérer les dependances préconfigurer pour éviter de faire de la configuration manuelle - Une fois installer les dépendances sont sauvegarder dans la package.json, config.js

Configurer des dépendances

Principales propriétés qui peuvent être surchargés:

  • main
  • map
  • format

jspm install github:user/repo -o "{ main: 'path/to/main' }"
